# Code Companion

A high-end, feature-rich coding learning platform built with a modern React stack. Master algorithms through curated problems organized by patterns, track your progress, and level up your skills.

- **Personalized Dashboard**: Track your problem-solving statistics, streaks, and completion rates.
- **Pattern-Based Learning**: Categorized coding problems (e.g., Two Pointer, Sliding Window) to build algorithmic intuition.
- **Data Visualization**: Interactive charts showing your progress by difficulty and pattern.
- **Gamification**: Real-time leaderboards and submission history.
- **Premium UI**: Modern dark mode with glassmorphism, smooth animations, and a responsive layout.
- **Admin Dashboard**: Comprehensive interface for managing coding content and patterns.
